# ابزار‌ها 🧮

یک اپلیکیشن وب پیشرونده (PWA) شامل محاسبه‌گر مالی و مجموعه‌ای از ابزارهای کاربردی روزمره — کاملاً آفلاین، بدون بک‌اند، فقط HTML/CSS/JS خالص.

## ابزارهای موجود
- **محاسبات کاربردی:** BMI، محاسبه سن (شمسی)، تبدیل تاریخ (شمسی/میلادی)، درصدگیری، تخفیف و مالیات
- **مالی و روزمره:** وام و اقساط، سود سپرده بانکی (ساده/مرکب)، تقسیم صورت‌حساب، سود و زیان معامله، فاصله بین دو تاریخ/ساعت، تورم، قیمت متری
- **QR و امنیت:** ساخت QR (متن/وای‌فای) + دانلود، اسکن QR از روی عکس، رمز عبور قوی، رمزنگاری سبک متن
- **متفرقه:** امضای دیجیتال (خروجی PNG شفاف)، پاک‌سازی/مرتب‌سازی لیست، نمایش اطلاعات IP

## اجرای محلی
هر سروری کافیست، مثلاً:
```bash
cd toolbox
python3 -m http.server 8080
```
سپس آدرس `http://localhost:8080` را باز کنید.

> نکته: برای نصب PWA (Add to Home Screen) و کارکرد Service Worker، باید از طریق `http://localhost` یا `https` باز شود — نه با باز کردن مستقیم فایل (`file://`).

## انتشار روی GitLab Pages
1. یک پروژه جدید در GitLab بسازید، مثلاً به اسم `abzarha`.
2. محتوای این پوشه را داخل ریپو کپی کنید و پوش کنید:
   ```bash
   cd toolbox
   git init
   git remote add origin git@gitlab.com:USERNAME/abzarha.git
   git add .
   git commit -m "اولین نسخه ابزار‌ها"
   git push -u origin main
   ```
3. فایل `.gitlab-ci.yml` (همراه پروژه) به‌طور خودکار سایت را روی GitLab Pages منتشر می‌کند.
4. بعد از اتمام پایپ‌لاین، آدرس سایت این شکلی خواهد بود:
   `https://USERNAME.gitlab.io/abzarha/`
5. برای نصب اپ روی گوشی: آدرس بالا را در مرورگر (Chrome/Safari) باز کنید و گزینه‌ی «افزودن به صفحه اصلی / Install App» را بزنید.

## ساختار پروژه
```
toolbox/
├── index.html
├── manifest.json         ← تنظیمات PWA
├── sw.js                 ← Service worker (کش آفلاین)
├── css/style.css
├── js/
│   ├── jalali.js          ← تبدیل تاریخ شمسی/میلادی
│   ├── tools.js           ← لیست دسته‌ها و ابزارها
│   └── app.js             ← روتر + منطق همه‌ی ابزارها
└── icons/
```

## افزودن ابزار جدید
1. یک آیتم جدید به آرایه‌ی `TOOLS` در `js/tools.js` اضافه کنید (id, cat, name, desc, icon).
2. در `js/app.js` یک تابع به‌نام همان `id` داخل شیء `RENDERERS` بنویسید که HTML و منطق محاسبه را می‌سازد.

## یادداشت
- ابزارهای «نمایش اطلاعات IP»، «ساخت QR» و «اسکن QR» برای کارکرد کامل به اتصال اینترنت نیاز دارند (به دلیل استفاده از سرویس ipapi.co و کتابخانه‌های CDN). بقیه‌ی ابزارها کاملاً آفلاین کار می‌کنند.
- رمزنگاری متن (`textlock`) یک روش سبک XOR+Base64 است، نه استاندارد رمزنگاری امنیتی؛ فقط برای مخفی‌سازی ساده مناسب است.
