# ابزار‌ها 🧮

یک اپلیکیشن وب پیشرونده (PWA) شامل محاسبه‌گر مالی و مجموعه‌ای از ابزارهای کاربردی روزمره — کاملاً آفلاین، بدون بک‌اند، تک‌فایل (فقط HTML/CSS/JS خالص داخل یک `index.html`).

## امکانات کلی
- **ماشین‌حساب** با صفحه‌نمایش اختصاصی، تاریخچه‌ی محاسبات و کپی/جای‌گذاری سریع (این صفحه‌نمایش فقط تو تب ماشین‌حساب دیده می‌شه، نه تو تب ابزارها).
- **جست‌وجوی ابزار** با کلیدواژه‌های فارسی و انگلیسی.
- **دسته‌بندی قابل ویرایش:** ساخت دسته‌بندی دلخواه، تغییرنام/حذف، و جابه‌جایی ابزارها بین دسته‌ها با کشیدن (Drag & Drop).
- **میان‌برهای پرکاربرد:** پین‌کردن ابزار دلخواه با ⭐ برای دسترسی سریع بالای صفحه.
- **آفلاین واقعی:** با سرویس‌ورکر ثبت‌شده، بعد از اولین بازدید همه‌چیز کش می‌شه و بدون اینترنت هم کار می‌کنه (به‌جز ابزارهایی که ذاتاً به سرویس بیرونی نیاز دارن).
- **نشانگر آنلاین/آفلاین** بالای صفحه و هشدار مخصوص تو ابزارهایی که به اینترنت نیاز دارن.
- **بررسی خودکار نسخه جدید:** با نصب نسخه‌ی جدید فایل‌ها روی سرور، یه نوار «نسخه‌ی جدید موجوده» بالای صفحه ظاهر می‌شه.
- چند پوسته‌ی رنگی (تیره/روشن/و چند تم دیگه) و اندازه‌ی فونت قابل تنظیم.

## دسته‌بندی‌های پیش‌فرض و ابزارها

**🧮 محاسبات کاربردی**
BMI (شاخص توده بدنی)، محاسبه سن (شمسی)، تبدیل تاریخ جامع (شمسی/میلادی/قمری)، درصدگیری، تخفیف و مالیات، مبدل واحدها (طول، وزن، حجم، دما، حافظه، سرعت اینترنت، آشپزی)

**💰 مالی و روزمره**
وام و اقساط، سود سپرده بانکی، تقسیم صورت‌حساب گروهی با اسم و سهم نابرابر (دنگی)، سود و زیان معامله، فاصله بین دو تاریخ/ساعت، تورم و کاهش ارزش پول، قیمت متری و ساختمانی، حباب سکه و طلا، محاسبه مهریه سکه‌ای

**🔐 QR و امنیت**
ساخت QR (وبسایت/متن/وای‌فای/پیامک/تماس/واتساپ/اینستاگرام/تلگرام/کارت‌ویزیت/ایمیل/لوکیشن/رویداد تقویم) با رنگ و لوگوی دلخواه + دانلود، اسکن زنده‌ی QR با دوربین، رمز عبور قوی (تا ۵۰۰ کاراکتر) + بررسی سلامت رمز موجود، مخفی‌سازی متن در عکس (استگانوگرافی)، رمزنگاری واقعی متن با AES-256

**🛠️ ابزارهای متفرقه**
امضای دیجیتال (خروجی PNG شفاف)، پاک‌سازی و مرتب‌سازی لیست، نمایش اطلاعات IP، استخراج متن از عکس (OCR)، کد مورس (فارسی و انگلیسی، با پخش صدا و راهنمای آموزشی)، چک‌لیست‌های آماده (سفر، کمپینگ، سرویس خودرو) + آیتم دلخواه

> کاربر می‌تونه از تب «ابزارها» دسته‌بندی جدید بسازه و ابزارها رو بین دسته‌ها جابه‌جا کنه؛ لیست بالا فقط چینش پیش‌فرضه.

## اجرای محلی
هر سروری کافیست، مثلاً:
```bash
python3 -m http.server 8080
```
سپس آدرس `http://localhost:8080` را باز کنید.

> نکته: برای نصب PWA (Add to Home Screen) و کارکرد Service Worker، باید از طریق `http://localhost` یا `https` باز شود — نه با باز کردن مستقیم فایل (`file://`).

## انتشار روی GitLab Pages
1. یک پروژه جدید در GitLab بسازید، مثلاً به اسم `abzarha`.
2. فایل‌های این پوشه را داخل ریپو کپی کنید و پوش کنید:
   ```bash
   git init
   git remote add origin git@gitlab.com:USERNAME/abzarha.git
   git add .
   git commit -m "بروزرسانی ابزار‌ها"
   git push -u origin main
   ```
3. فایل `.gitlab-ci.yml` (اگه همراه پروژه باشه) به‌طور خودکار سایت را روی GitLab Pages منتشر می‌کند؛ در غیر این صورت کافیه یه پایپ‌لاین ساده‌ی «کپی فایل‌ها به `public/`» بسازید.
4. بعد از اتمام پایپ‌لاین، آدرس سایت این شکلی خواهد بود:
   `https://USERNAME.gitlab.io/abzarha/`
5. برای نصب اپ روی گوشی: آدرس بالا را در مرورگر (Chrome/Safari) باز کنید و گزینه‌ی «افزودن به صفحه اصلی / Install App» را بزنید.

## ساختار پروژه
```
abzarha/
├── index.html         ← کل اپ (ساختار + استایل + منطق همه‌ی ابزارها)
├── sw.js               ← Service worker (کش آفلاین + تشخیص نسخه‌ی جدید)
├── manifest.json       ← تنظیمات PWA (نام، آیکون، تم رنگی)
├── icon-192.png        ← آیکون اپ (۱۹۲×۱۹۲)
└── icon-512.png        ← آیکون اپ (۵۱۲×۵۱۲)
```
> `manifest.json` و دو فایل آیکون باید حتماً کنار `index.html` و `sw.js` روی سرور باشن؛ سرویس‌ورکر همه‌ی این فایل‌ها رو در نصب اولیه کش می‌کنه و اگه یکیشون گم باشه، نصب سرویس‌ورکر (و درنتیجه حالت آفلاین) ممکنه با خطا مواجه بشه.

## افزودن ابزار جدید
همه‌چیز داخل `index.html` است؛ برای اضافه‌کردن یک ابزار جدید:
1. یک آیتم به آرایه‌ی `toolsIndex` اضافه کنید (`id`, `icon`, `color`, `name`, `desc`, `kw`).
2. آیدی ابزار را در `DEFAULT_TOOL_CAT` به یکی از دسته‌های پیش‌فرض نگاشت کنید (یا بگذارید خودکار برود توی «متفرقه»).
3. یک ورودی متناظر در `TOOL_META` اضافه کنید (برای نمایش صحیح در میان‌برهای پین‌شده).
4. یک بلوک `<div class="toolpage" id="tool-{id}">…</div>` برای HTML و فرم‌های ابزار بنویسید.
5. توابع محاسبه/منطق ابزار را در تگ `<script>` اضافه کنید و در صورت نیاز به مقداردهی اولیه، یک شرط برای `name=='{id}'` داخل تابع `openTool()` اضافه کنید.
6. نسخه‌ی `CACHE_NAME` در `sw.js` را یکی بالا ببرید تا کاربرها نسخه‌ی جدید را (به‌جای کش قدیمی) بگیرند.

## یادداشت
- ابزارهای «نمایش اطلاعات IP» و «استخراج متن از عکس (OCR)» (فقط بار اولی که موتورش دانلود می‌شه) برای کارکرد کامل به اتصال اینترنت نیاز دارند؛ اپ خودش تشخیص می‌ده آفلاینی یا نه و هشدار مخصوص نشون می‌ده. بقیه‌ی ابزارها کاملاً آفلاین کار می‌کنند.
- رمزنگاری متن با AES-256-GCM واقعی (از طریق Web Crypto API مرورگر) انجام می‌شود و برای اطلاعات حساس هم قابل‌اعتماده؛ فقط توجه کن که اگر رمز عبورش رو فراموش کنی، هیچ راهی برای بازیابی متن وجود نداره.
- محاسبه‌ی «مهریه» فقط برای مهریه‌ی سکه‌ای (امامی/بهار آزادی/نیم/ربع سکه) است و بر پایه‌ی قیمت روزی که خودتان در تنظیمات وارد می‌کنید محاسبه می‌شود؛ برای مراحل قانونی و اجراییات رسمی باید از منابع رسمی استعلام گرفت.
- جدول «مورس فارسی» طوری طراحی شده که هر حرف فارسی کد یک حرف لاتین را قرض می‌گیرد (استاندارد رایج بین‌المللی برای الفبای غیرلاتین)؛ برای تبدیل درست، حتماً زبان مناسب را قبل از تبدیل انتخاب کنید.
