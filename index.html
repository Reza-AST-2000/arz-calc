نرخ‌های این بخش نمونهٔ قابل‌ویرایش‌اند. با «دریافت خودکار» تلاش می‌شود از یک سرویس رایگان با محدودیت درخواست دریافت شود (ممکن است گاهی در دسترس نباشد). برای دقت بیشتر می‌توانید کلید API رایگان خودتان را از brsapi.ir بگیرید و در تنظیمات ⚙ وارد کنید. مقادیرِ ویرایش‌شده روی همین گوشی ذخیره می‌شوند.

کلید API اختصاصی (اختیاری)

برای دریافت پایدارتر نرخ بازار آزاد، می‌توانید کلید رایگان خودتان را از brsapi.ir بگیرید و اینجا وارد کنید. در غیر این‌صورت از کلید نمایشی عمومی استفاده می‌شود که ممکن است محدود باشد.
